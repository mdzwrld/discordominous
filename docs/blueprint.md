# **App Name**: Discord Dominus

## Core Features:

- High-Impact Sales Page UI: Display the provided compelling copy and structured sections (Hero, Dor, Verdade, Amador vs Operador, etc.) across the site to guide the user towards conversion with maximum impact.
- Adaptive Responsive Design: Ensure the site is 100% responsive and mobile-first, with large, comfortable buttons and optimized text for reading across all devices.
- Sticky Navbar with Dynamic CTA: Implement a fixed navigation bar with the logo on the left and a prominent Call to Action button on the right. The navbar will initially be translucent and solidify upon scroll.
- Call to Action Handling: Manage the main 'QUERO CONSTRUIR MINHA MÁQUINA NO DISCORD AGORA' button, integrating payment and guarantee sections to drive immediate purchases.
- SEO & Performance Optimization: Configure all necessary SEO elements including meta tags, Open Graph, a single H1, and strategic meta descriptions to ensure optimal search engine visibility and fast loading times.
- Discord Strategy Personalizer (AI Tool): An optional interactive section where users can input basic information about their business goals and receive a concise, AI-generated tip or conceptual starting point related to building a Discord community for monetization.

## Style Guidelines:

- Background color: The primary background is a deep, sophisticated dark blue-black (#0B0F14), complemented by a slightly lighter dark blue (#111827) for secondary backgrounds, enhancing the premium dark mode aesthetic.
- Primary action/highlight color: A vibrant neon green (#00F5A0) is used for impactful elements like Call to Action buttons, critical highlights, and subtle glows, conveying energy and modernity. Its hue (155°) is selected for strong contrast and visibility.
- Accent color: An alternative neon blue (#00C2FF) adds versatility for complementary highlights or visual distinctions. Its hue (200°) offers a distinct yet harmonious contrast to the primary highlight.
- Text color: A light, off-white (#E5E7EB) provides excellent readability against the dark backgrounds, maintaining a clean and professional appearance.
- Font family: 'Inter' (sans-serif) is chosen for its modern, clean, and highly readable characteristics across all weights. Headline text will use Extra Bold weight for maximum impact, subheadlines Semi Bold, body text Regular for clear readability, and CTAs Bold to stand out. Note: currently only Google Fonts are supported.
- Use minimalist, clean icons, especially for listing features ('O Método' and 'O Que Está Incluso'), to complement the sophisticated dark mode. The style should feel modern and sharp.
- A clean, aggressive, and ample-spaced layout emphasizes clarity and impact. Cards feature rounded borders (12px) and soft shadows. A fixed navbar with a logo and CTA, paired with generous section spacing, maintains user focus and a premium feel.
- Subtle, smooth animations enhance the user experience without distraction. This includes a gentle glow effect on buttons, a slight scale (1.03) on hover for interactive elements, smooth scrolling throughout the page, and light animations as elements enter the viewport to add fluidity.
